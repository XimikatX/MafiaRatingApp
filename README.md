# MafiaRatingApp
Student project application made with Kotlin and Jetpack Compose

Implemented:
* Careful MVVM architecture
* Game construction screen with multiple states to omit Navigation mess
* Bottom navigation bar with two screens 
* Donut chart for statistics visualization
